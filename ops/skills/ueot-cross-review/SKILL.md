# UEOT Cross Review

Use this skill when producing a review in `council/reviews/`.

## Required Fields

- `from`
- `to`
- `relation`
- `claim`
- `impact`
- `required_action`

## Review Discipline

- critique claims, not personalities
- preserve the target domain's authorship boundary
- make the action request concrete
