# Omega Council Cycle

The architect runs this pass with all domains in mind.

1. Review the newest accepted memos across all domains.
2. Identify at least:
   - one convergent pattern
   - one contradiction
   - one missing bridge
3. Emit cross-domain follow-up tasks.
4. Update relevant chapter dossiers to reflect new integration points.

Output targets:

- `council/reviews/`
- `council/resolutions/`
- `book/chapters/`
