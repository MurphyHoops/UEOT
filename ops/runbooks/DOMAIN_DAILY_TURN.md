# Domain Daily Turn

For the assigned domain agent:

1. Read the constitution files.
2. Read your domain `STATE.md`.
3. Read the latest accepted memo in your domain.
4. Inspect cross-domain reviews involving your domain.
5. Produce one of the following:
   - a new draft memo
   - a revision to a draft memo
   - a cross-domain review note
   - a state update reflecting a resolved issue

Constraints:

- stay inside your domain or council review space
- do not alter constitutional files
- do not silently overwrite accepted claims
- produce at least one cross-domain bridge candidate

Default output target:

- `domains/<domain>/memos/` for a memo
- `council/reviews/` for a review
- `domains/<domain>/STATE.md` for accepted-state updates when explicitly justified
