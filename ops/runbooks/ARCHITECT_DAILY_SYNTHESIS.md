# Architect Daily Synthesis

1. Read new memos and reviews from all domains.
2. Detect contradictions, overlap, and bridge opportunities.
3. Issue resolutions where necessary.
4. Update chapter dossiers affected by accepted outcomes.
5. Update constitutional or method documents only if a formal change decision
   exists.
6. Record the day's synthesis in shared memory.

Constraints:

- no silent canon drift
- every ruling must be written
- unresolved tensions must remain visible
